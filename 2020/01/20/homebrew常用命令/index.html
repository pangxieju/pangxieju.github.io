# Mac Homebrew 常用命令

# Homebrew

# 简介

Homebrew 是 Mac 平台下的包管理工具。我们可以通过 Homebrew 方便的安装和管理软件。

# 官方地址

https://brew.sh

# 自身相关

# 帮助信息

brew help

# 查看版本

brew -v

# 自更新

brew update

# 卸载 Homebrew

cd `brew --prefix`
rm -rf Cellar
brew prune
rm `git ls-files`
rm -r Library/Homebrew Library/Aliases Library/Formula Library/Contributions
rm -rf .git
rm -rf ~/Library/Caches/Homebrew

# 软件包管理相关

在 brew 后面加上 cask 一般管理是的 GUI 软件包（如钉钉，Sourcetree等），例如：

brew cask install dingtalk

# 搜索安装包

brew search [包名]      # 搜索包

# 安装包

brew install [包名]     # 安装指定包
brew install -f [包名]  # -f 覆盖安装

# 卸载安装包

brew uninstall [包名]   # 卸载指定包

# 升级安装包

brew upgrade            # 升级所有包
brew upgrade [包名]      # 升级指定包

# 查看已安装包

brew list

# 查看已安装可更新包

brew outdated

# 清理安装包

brew cleanup            # 清理所有旧版本包
brew cleanup [包名]      # 清理指定旧版本包
brew cleanup -n          # 查看可清理旧版本包
